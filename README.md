## Currently

- Building **OpenSec**, an AI-native OSS security platform for vulnerability detection, triage, remediation, and secure code intelligence.
- Working on **AEGIS**, a repository-scale vulnerability detection and remediation system using SAST signals, RAG code retrieval, program-flow reasoning, exploitability analysis, multi-agent triage, and automated patch generation.

## OpenSec

OpenSec focuses on moving vulnerability research from noisy alerts to validated, actionable fixes.

| Area | Work |
| --- | --- |
| Detection | SAST signals, repository analysis, code retrieval, CWE/CVE mapping |
| Reasoning | Program-flow analysis, exploitability assessment, multi-agent triage |
| Remediation | Patch generation, fix validation, evidence-backed recommendations |
| Data | TRACEVUL: 94,759 canonical records, 184K training samples, 19+ languages, 553 CWEs, 6,192 OSS repositories |
| Research | 28 zero-day vulnerability candidates across 6 OSS repositories, including credited discovery of CVE-2026-42766 in OpenSSL |

OpenSec received an **INR 38 lakh Government of India grant** for AI-native vulnerability detection and automated remediation.

## Selected Work

**AEGIS**  
Agentic vulnerability detection and remediation for open-source repositories. Combines SAST, RAG, program-flow reasoning, exploitability analysis, and patch generation.

**TRACEVUL**  
Vulnerability intelligence corpus connecting CVEs to code regions, vulnerable/fixed states, call graphs, patch evidence, root causes, and validation metadata.

**CloudChase**  
U-Net based cloud coverage prediction using multi-spectral INSAT-3DS imagery, physics-aware losses, SSIM, PSNR, and nowcasting workflows.  
[Repository](https://github.com/mayank-jangid-moon/CloudChase)

**TxnGuard**  
Secure banking, autonomous fraud detection, and AML multi-agent framework using vectorized transaction evidence and CrewAI triage.

**Virtual Try-On**  
Pose-guided try-on with OpenPose, SCW-VTON style warping, skin-tone analysis, and visual retrieval over 7K+ catalog items.  
[Repository](https://github.com/Kushalkhemka/fashion-vton)

**LeetCode**  
C++ solutions focused on efficient approaches and implementation clarity.  
[Repository](https://github.com/Kushalkhemka/leetcode)
